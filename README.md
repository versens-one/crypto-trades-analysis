# Crypto Trades Analysis
Analysis of cryptocurrency trading activity using SQL and Python

This project analyzes cryptocurrency trading activity using SQL and Python.
The goal is to explore trading patterns, trading volume distribution, and trader behavior in crypto markets.

The dataset contains cryptocurrency trades with the following fields:
timestamp — time of the trade
symbol — trading pair (BTC/USDT, ETH/USDT etc.)
price — price of the asset
quantity — amount traded
side — buy or sell
trade_volume_usd — trade size in USD

# Objectives

The analysis focuses on several key questions:

Which cryptocurrencies have the highest trading volume
How trading activity changes over time
What is the average trade size
Distribution of small vs large trades
Buy vs sell balance in the market
